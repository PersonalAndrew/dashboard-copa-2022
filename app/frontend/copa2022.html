Projeto

Metodologia e Créditos

Este dashboard foi desenvolvido para transformar dados brutos de partidas da Copa do Mundo 2022 em visualizações interativas voltadas à análise de desempenho no futebol.

📊

Fonte dos dados

Os dados utilizados no projeto têm como base o conjunto público da StatsBomb, contendo informações de partidas, eventos, jogadores, escalações, chutes, passes e ações em campo.

⚽

O que é analisado

O dashboard permite analisar resultados, campanha do Brasil, artilheiros, rankings por equipe, escalações, finalizações, passes, mapas de calor, posições médias, momentos do jogo e linha do tempo dos eventos.

🧠

Resumo inteligente

A seção de resumo inteligente interpreta os principais padrões da partida, destacando volume ofensivo, criação de chances, domínio territorial, eficiência e jogadores de maior impacto.

⭐

Ranking individual

O ranking individual é um índice próprio do dashboard. Ele não representa uma nota oficial da StatsBomb, mas uma métrica criada para combinar participação, gols, xG, passes, dribles, ações ofensivas e influência no jogo.

🛠️

Tecnologias utilizadas

O projeto foi construído com Python, FastAPI, SQLite, HTML, CSS, JavaScript e Chart.js, integrando backend, banco de dados e visualizações interativas em uma aplicação web.

👤

Autoria

Desenvolvido por Andrew Gouveia como projeto de Análise de Desempenho e Ciência de Dados aplicados ao futebol, com foco em visualização, interpretação e comunicação de dados esportivos.

Observação: este projeto tem finalidade educacional e de portfólio. As métricas próprias foram criadas para fins de estudo, interpretação e apresentação visual dos dados.
